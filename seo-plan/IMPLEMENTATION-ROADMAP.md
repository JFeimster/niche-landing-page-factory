# Implementation Roadmap

Project: `distilledfunding.com`  
Date: 2026-03-27

## Phase 1: Foundation (Weeks 1-4)

Goals:
- tighten technical SEO baseline
- maximize performance of created verticals
- create reusable publish standards for future pages

Actions:
- finalize metadata and canonical handling for `/[vertical]` pages
- enforce `noindex` for incomplete drafts
- add schema components (`Organization`, `WebPage`, `FAQPage` where applicable)
- expand `gig-worker-loans` and complete `electricians` quality gate
- set tracking dashboard:
  - organic sessions by page
  - CTA click-through rate
  - call booking completions

Exit criteria:
- 2 high-quality indexable vertical pages live
- technical checks pass for robots/sitemap/canonicals
- dashboard reporting active

## Phase 2: Expansion (Weeks 5-12)

Goals:
- launch first major wave of future verticals
- establish repeatable hub-and-spoke cadence

Actions:
- publish wave 1 hubs: contractors, hvac, plumbers, roofers, auto-repair
- publish 10-12 spoke blogs mapped to those hubs
- implement internal linking template:
  - hub to spoke
  - spoke to hub
  - sibling hub links by funding type
- add quarterly content pruning rules for thin pages

Exit criteria:
- at least 7 indexable vertical hubs total
- 15+ spoke pages live
- stable crawl/indexation with low invalid URL growth

## Phase 3: Scale (Weeks 13-24)

Goals:
- build topical authority across service-business clusters
- improve rankings for high-intent commercial queries

Actions:
- publish wave 2 and wave 3 verticals by performance priority
- launch comparison pages and decision guides
- improve CWV for heavy templates and media usage
- begin outreach/link earning with data-backed guides/tools

Exit criteria:
- 12+ active vertical hubs
- top-10 rankings for long-tail terms in at least 5 verticals
- qualified organic leads growth trend month-over-month

## Phase 4: Authority (Months 7-12)

Goals:
- sustain rankings and improve brand trust signals
- convert authority into durable lead flow

Actions:
- expand expert-authored content and editorial review notes
- publish benchmark/insight reports from anonymized funnel data
- enrich structured data footprint where valid and useful
- run quarterly architecture optimization and internal link audits

Exit criteria:
- 15+ quality-controlled vertical hubs
- strong top-10 footprint across target clusters
- predictable organic lead contribution to pipeline

## Dependencies and Risks

Dependencies:
- editorial capacity for 2-4 posts/week
- engineering support for template-level SEO updates
- analytics ownership for conversion attribution

Risks:
- quality inconsistency as page count scales
- internal linking debt
- indexation of low-value pages

Mitigation:
- enforce publish checklist
- add pre-publish QA review
- maintain explicit `noindex` defaults for incomplete pages
